# Aula 7

## Operadores Aritméticos

- `+` Adição
- `-` Subtração
- `*` Multiplicação
- `/` Divisão
- `**` Potência
- `//` Divisão Inteira
- `%` Resto da Divisão (MODULO)

### Exemplo:

#### Adição
``5+2==7``

#### Subtração
``5-2==3``

#### Multiplicação
``5*2==10``

#### Divisão
``5/2==2.5``

#### Potência
``5**2==25``

#### Divisão Inteira
``5//2==2``

#### Resto da Divisão
``5%2==1``

## Ordem de Precedência

1. `()`
2. `**`
3. `*` `/` `//` `%`
4. `+` `-`

### Exemplos:

* ```5+3*2==11```
* ```3*5+4**2==31```
* ```3*(5+4)**2==243```
